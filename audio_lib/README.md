# Audio_lib

A library/API to easily manage audio tracks on Luanti, featuring accessibility options. Natively supports Minetest Game sounds.  


<a href="https://liberapay.com/Zughy/"><img src="https://i.imgur.com/4B2PxjP.png" alt="Support my work"/></a>  

### How To
Have a look at the [DOCS](DOCS.md)

### Known conflicts
Any mod that overrides `core.sound_play`, adding new parameters (done here to natively support MTG sounds)

### Commands
* `/audiosettings`: open the audio settings menu (or use the built-in item)
